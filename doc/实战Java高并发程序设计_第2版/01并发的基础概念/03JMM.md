#1、原子性（ Atomicity)
    原子性是指一个操作是不可中断的。即使是在多个线程一起执行的时候，一个操作一
    旦开始，就不会被其他线程干扰。 
#2、可见性（ Visibility)
    可见性是指当一个线程修改了某一个共享变量的值时，其他线程是否能够立即知道这个修改。
#3、有序性（ odering)
    有序性问题的原因是程序在执行时，可能会进行指令重排，重排后的指令与原指令的顺序未必一致。
